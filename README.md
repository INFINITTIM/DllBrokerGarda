# Message Broker

Библиотека на С++ для эффективного обмена сообщениями между различными модулями.


## Компоненты

### Event (`include/Event.hpp`)

Структура для передачи данных между модулями.

| Поле | Тип | Описание |
|------|-----|----------|
| `type` | `uint32_t` | Идентификатор типа события |
| `message` | `std::string` | Сообщение (описание события) |

### Manager (`include/Manager.hpp`)

Центральный диспетчер сообщений.

| Метод | Параметры | Описание |
|-------|-----------|----------|
| `add_module` | `Module*` | Регистрация модуля в системе |
| `add_sub_module` | `Module*`, `EventTypeID` | Добавление подписки модуля на тип события |
| `remote_sub_module` | `Module*`, `EventTypeID` | Удаление подписки |
| `broadcast` | `Module*`, `const Event&` | Рассылка события всем подписчикам |

### Module (`include/Module.hpp`)

Базовый класс для создания пользовательских модулей.

| Метод | Параметры | Описание |
|-------|-----------|----------|
| `subscribe` | `EventTypeID` | Подписка на тип события |
| `unsubscribe` | `EventTypeID` | Отписка от типа события |
| `send` | `EventTypeID`, `const std::string&` | Отправка события (сообщения) |
| `deliver` | `const Event&` | Добавление события во входящую очередь |
| `read_messages` |  | Обработка входящей очереди сообщений |
